== Installation ==

Installing Query Manager is just like installing most other plugins. [Check out the codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins) if you have any questions.

After installation, you can find the Query Manager settings panel under the main Settings menu item in your admin panel.
